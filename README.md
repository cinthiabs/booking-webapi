# About this Project 
This project was generated with `.Net 8.0`. It is a Web API for creating, reading, updating, and deleting bookings records. 

# Pre-requirements
To clone and run this project, you will need:
 - Visual Studio 2022 IDE or  Visual Studio Code

# Running this Project 
In this project, you can run `dotnet run` and navigate to `http://localhost:5115`

# API Endpoints
The API provides the following endpoints: 
    
    GET /api/v1/Booking 
    POST /api/v1/Booking
    PUT /api/v1/Booking
    DELETE /api/v1/Booking
    GET /api/v1/Booking/{id}
